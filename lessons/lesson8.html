Сабақ 8: Циклдер

Теория

Циклдер — белгілі бір код бөлігін бірнеше рет қайталап орындауға мүмкіндік береді.

Python тілінде екі негізгі цикл бар: for және while.

for циклі

for циклі белгілі бір тізім, диапазон немесе жол сияқты элементтер тізбегін қайталайды.

for i in range(5):
    print("Сан:", i)
while циклі

while циклі шарт ақиқат болғанша қайталанады.

x = 0
while x < 5:
    print("x мәні:", x)
    x += 1
break және continue

break — циклді толық тоқтатады, continue — ағымдағы қадамды өткізіп, келесі қадамға өтеді.

for i in range(5):
    if i == 3:
        break
    print(i)
Жаттығу
  1. 1-ден 10-ға дейінгі сандарды шығаратын for циклін жазыңыз.
  2. 1-ден 5-ке дейінгі сандарды шығаратын while циклін жазыңыз.
  3. 10-нан 1-ге дейінгі сандарды кері бағытта шығаратын цикл құрыңыз.
  4. 0-ден 20-ға дейінгі жұп сандарды шығаратын цикл жазыңыз.
  5. 5 сан енгізіп, олардың қосындысын шығаратын бағдарлама құрыңыз.
