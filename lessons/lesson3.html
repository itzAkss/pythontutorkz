Сабақ 3: Операциялар мен өрнектер

Теория

Python тілінде сандармен жұмыс істеу үшін әртүрлі операциялар қолданылады. Операциялар — бұл есептеулерді орындайтын таңбалар.

Негізгі арифметикалық операциялар:
  • + — қосу
  • - — азайту
  • * — көбейту
  • / — бөлу (нәтиже бөлшек сан болады)
  • // — бүтін бөлу (қалдықсыз)
  • % — қалдық табу
  • ** — дәреже (мысалы, 2**3 = 8)
Мысал:
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3
print(a % b)   # 1
print(a ** b)  # 1000

Өрнектер — бұл айнымалылар мен операциялардан құралған есептеулер.

Жаттығу

Екі сан енгізіп, оларды қосу, азайту, көбейту және бөлуді экранға шығаратын бағдарлама жазыңыз.
