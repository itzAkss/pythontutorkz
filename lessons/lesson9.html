Сабақ 9: Функциялар

Теория

Функциялар — белгілі бір тапсырманы орындайтын код блогы.

Функциялар кодты бірнеше рет қолдануға және бағдарламаны ықшам әрі түсінікті етуге көмектеседі.

Функцияны анықтау
def salem_ber():
    print("Сәлем!")

salem_ber()  # Функцияны шақыру
Параметрлері бар функция
def kos(a, b):
    print(a + b)

kos(3, 5)  # 8
Қайтарылатын мәні бар функция
def kobeytu(a, b):
    return a * b

nateje = kobeytu(4, 6)
print(nateje)  # 24
Жаттығу
  1. Сәлемдесу мәтінін шығаратын функция жазыңыз.
  2. Екі санды қосатын функция құрыңыз.
  3. Берілген санның квадратын қайтаратын функция жазыңыз.
  4. Пайдаланушы енгізген 3 санның орташа мәнін есептейтін функция жазыңыз.
  5. Сөзді параметр ретінде қабылдап, оның ұзындығын қайтаратын функция жазыңыз.
