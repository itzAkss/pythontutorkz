Сабақ 4: Шартты операторлар (if/else)

Теория

Шартты операторлар белгілі бір шартқа байланысты әртүрлі әрекет орындауға мүмкіндік береді.

Python тілінде негізгі шартты операторлар:

  • if — егер шарт орындалса
  • else — әйтпесе
  • elif — әйтпесе егер (қосымша шарт)
Мысал:
age = 18

if age >= 18:
    print("Сіз кәмелет жасына толдыңыз")
else:
    print("Сіз әлі кәмелет жасына толмадыңыз")

Бұл бағдарлама адамның жасына қарай хабарлама шығарады.

Қосымша мысал (elif):
grade = 85

if grade >= 90:
    print("Өте жақсы")
elif grade >= 70:
    print("Жақсы")
else:
    print("Тырысу қажет")
Жаттығу

Пайдаланушы енгізген сан 0-ден үлкен, кіші немесе тең екенін тексеретін бағдарлама жазыңыз.
