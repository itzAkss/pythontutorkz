Сабақ 1: Python-ға кіріспе

Теория

Python — әлемдегі ең танымал бағдарламалау тілдерінің бірі.

Оны қолдануға болады:

  • веб-сайттар жасауға
  • ойын жасауға
  • жасанды интеллект құруға
  • деректерді талдауға

Python басқа тілдерден қарағанда оңай оқылады және түсінікті.

Python кодын жазуға арналған арнайы бағдарламаларды IDE (мысалы: VS Code, PyCharm) деп атайды.

Мысал:
print("Сәлем, Әлем!")

Бұл бағдарлама экранға "Сәлем, Әлем!" деп шығарады.
Бұл — Python-дағы ең алғашқы бағдарлама.
