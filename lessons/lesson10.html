Сабақ 10: Модульдер және кітапханалар

Теория

Модульдер — Python кодын бөлек файлдарға бөліп, қайта қолдануға мүмкіндік береді.

Кітапханалар — көптеген дайын модульдердің жиынтығы. Python-да мыңдаған дайын кітапханалар бар.

Дайын модульді пайдалану
import math

print(math.sqrt(16))  # 4.0
Модульден тек қажетті бөлікті импорттау
from math import sqrt

print(sqrt(25))  # 5.0
Өз модуліңізді құру

Мысалы, mening_modulim.py атты файлда:

def salem():
    print("Сәлем, модульден!")

Ал негізгі бағдарламада:

import mening_modulim

mening_modulim.salem()
Жаттығу
  1. math модулін қолданып, квадрат түбір табыңыз.
  2. random модулін қолданып, 1 мен 10 арасында кездейсоқ сан шығарыңыз.
  3. Екі санды көбейтетін функция жазып, оны жеке модульде сақтап, негізгі бағдарламада қолданыңыз.
  4. datetime модулін қолданып, ағымдағы күн мен уақытты шығарыңыз.
  5. Өз атыңызды шығаратын функция жазып, оны басқа файлдан шақырыңыз.
