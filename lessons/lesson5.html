Сабақ 5: Циклдер (while және for)

Теория

Циклдер — бірдей әрекеттерді бірнеше рет орындауға мүмкіндік береді.

Python тіліндегі негізгі циклдер:
  • while — шарт орындалғанша қайталанады
  • for — белгілі бір тізімдегі әр элемент үшін қайталанады
Мысал (while):
count = 1

while count <= 5:
    print("Сан:", count)
    count += 1

Бұл бағдарлама 1-ден 5-ке дейінгі сандарды шығарады.

Мысал (for):
for i in range(1, 6):
    print("Сан:", i)

Бұл бағдарлама да 1-ден 5-ке дейінгі сандарды шығарады, бірақ for циклін қолданады.

Жаттығу

1-ден 10-ға дейінгі сандарды экранға шығаратын бағдарлама жазыңыз (for немесе while циклін қолданып).
