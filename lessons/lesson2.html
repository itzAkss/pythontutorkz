Сабақ 2: Айнымалылар және деректер типтері

Теория

Айнымалы — бұл деректерді сақтау үшін қолданылатын атау.

Айнымалыға мән беру үшін = белгісі қолданылады.

Python тілінде бірнеше негізгі деректер типтері бар:

  • int — бүтін сандар (мысалы: 5, 42)
  • float — бөлшек сандар (мысалы: 3.14, 2.5)
  • str — мәтіндер (мысалы: "Сәлем")
  • bool — логикалық мәндер (True немесе False)
Мысал:
name = "Аружан"
age = 15
height = 1.65
student = True

print(name)
print(age)
print(height)
print(student)

Бұл бағдарлама 4 айнымалы құрып, олардың мәндерін экранға шығарады.

Жаттығу

Өз атыңызды, жасыңызды және бойыңызды айнымалыға жазып, оларды экранға шығаратын бағдарлама жазыңыз.
